## Definition: Induced Matrix Norm

If $\lVert\cdot\rVert$ is a norm on $\mathbb{K}^{n}$, we define the **induced matrix norm**

$$
\lVert A\rVert := \sup_{x \in \mathbb{K}^{n} \setminus \{0\}} \frac{\lVert Ax\rVert}{\lVert x\rVert}
$$

for all $A \in \mathbb{K}^{n\times n}$, for which we use the same notation $\lVert\cdot\rVert$ as for the norm on $\mathbb{K}^{n}$.
